Porta Lógica AND/E

A porta lógica AND é um circuito eletrônico que funciona como uma multiplicação lógica, produzindo uma saída em nível alto (1) somente se todas as suas entradas estiverem em nível alto (1). A saída é baixa (0) se qualquer uma das entradas para baixa (0). Essa porta é fundamental em circuitos digitais para realizar operações como comparação de bits ou controle de fluxo de dados.

Porta Lógica OR/OU

A porta lógica OR é um circuito eletrônico que funciona como uma adição lógica, produzindo uma saída em nível alto (1) se pelo menos uma de suas entradas estiver em nível alto (1). A saída é baixa (0) somente se todas as entradas estiverem em nível baixo (0). Essa porta é fundamental em circuitos digitais para realizar operações como comparação de bits ou controle de fluxo de dados.
